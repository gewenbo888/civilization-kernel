# Civilization Kernel · 文明内核

> Civilization runs like software.

A bilingual 8-layer reading of modern civilization as an operating system. Each layer depends on the one below it and programs the one above. When the bottom buckles, the top falls — slowly, then suddenly.

## The stack (base → top)

- **L0 · Physical** — atoms, climate, ore bodies
- **L1 · Energy** — wood → coal → oil → electricity → fission → renewables
- **L2 · Compute** — transistors, fabs, GPU clusters
- **L3 · Network** — fiber, satellites, settlement rails
- **L4 · Intelligence** — brains, libraries, LLMs
- **L5 · Coordination** — markets, contracts, supply chains
- **L6 · Governance** — constitutions, courts, central banks, protocols
- **L7 · Narrative** — religion, ideology, brands, feeds

## Sections (12)

- 00 Opening + Hero (6 baseline stats)
- 01 What is a civilization stack (interactive 8-layer tower)
- L0–L7: one section per layer (reading + 5 examples + indicator + mini-stack position + insight)
- 09 Failure cascade (timeline T=0 → T=1w + 6 historical case rows)
- 10 Next stack (10 layers, two marked NEW: Agents + Tokens)
- Sources + 11 Closing manifesto

## Stack (tech)

- Next.js 14 (App Router)
- Framer Motion for scroll reveals + animated counters
- Tailwind with **bespoke 8-color layer spectrum** (one accent per layer)
- Spectral display + Inter body + JetBrains Mono data + Noto Serif/Sans SC for Han
- All SVG / CSS visualizations — no R3F (kept tight)
- Bilingual EN/中 toggle (LanguageProvider, localStorage, browser-locale fallback)

## Data

Public ~2024 vintage: IEA, Epoch AI, ITU, Visa Onchain, World Bank, IMF, WTO, DataReportal, plus historical frameworks from Joseph Tainter and Jared Diamond. Rounded for clarity and labeled "indicative."
